## Floor Types

### Normal Floors (19 total)
Standard battle encounters. Defeat all enemies to proceed.

### Boss Floors (6 total)
- **Floor 6** - Blaze encounter
- **Floor 10** - Sentinel + Fizz Djinn
- **Floor 15** - Granite Djinn
- **Floor 20** - Overseer (Chapter 1 finale)
- **Floor 25** - Neptune Sovereign
- **Floor 30** - Celestial Zeus (Tower finale)

### Rest Floors (5 total)
Floors 4, 8, 12, 16, 24, 28 - Allow loadout changes between battles.

## Progression Pattern

```
[Normal] → [Normal] → [Normal] → [REST]
[Normal] → [BOSS] → [Normal] → [REST]
... repeating with increasing difficulty ...
```

Rest floors appear every 4 floors, allowing strategic loadout adjustments before the next difficulty spike.
